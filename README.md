CarbonSense – The Smart Sustainability Companion

CarbonSense is an AI-powered sustainability platform that helps users track, understand, and reduce their carbon footprint.
It combines real-time carbon tracking, eco-route mapping, personalized AI recommendations, and gamified rewards to make sustainable living measurable and engaging.

Features:

User Profile System: Personalizes experience based on location, transport, and energy usage.

Carbon Tracker Dashboard: Monitors energy, water, travel, and CO₂ emissions.

Smart Eco-Route Mapping: Compares default and eco-friendly routes with CO₂ savings.

AI Sustainability Advisor: Provides tailored eco-tips, product suggestions, and green habits.

Gamified EcoPoints: Rewards positive actions with points, badges, and achievements.

Data Visualization: Displays weekly and monthly carbon trends.

Tech Stack:
Frontend: React, Vite, Tailwind CSS, Framer Motion, Recharts
Backend: Node.js, Express, Firebase or LocalStorage
APIs: OpenAI (GPT-4o-mini), OpenRouteService or MapBox Directions
Hosting: Replit with auto-sleep backend for cost efficiency

How It Works:

Users create a profile and log energy, travel, and waste data.

The dashboard calculates total CO₂ emissions using standard multipliers.

The AI Advisor gives personalized sustainability recommendations.

The Eco-Route Map suggests low-carbon travel routes.

EcoPoints reward users for positive actions and display progress badges.

Setup Instructions:

Clone or import this project into Replit.

Run npm install to install dependencies.

Add API keys in Replit Secrets:
OPENAI_API_KEY=<your_key>
ORS_API_KEY=<your_key>
FIREBASE_API_KEY=<optional>

Run npm run dev to start the frontend.

Optionally run node server/index.js to start the backend.
